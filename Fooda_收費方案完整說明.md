## 🎯 核心功能分析

### Fooda App 主要功能

| 功能模塊 | 技術實現 | 成本 |
|---------|---------|------|
| **AI 食物識別** | Google Gemini 1.5 Pro API | ~NT$ 0.008/次 |

| **歷史追蹤** | 雲端儲存 | ~NT$ 0.01/用戶/月 |
| **多語言支持** | i18n + AI 翻譯 | 固定成本 |

### 核心競爭力

✅ **AI 智能識別** - 自動辨識食物種類和份量  
✅ **多語言支持** - 中文、英文、日文全覆蓋  
✅ **精確營養分析** - 台灣食物資料庫  
✅ **簡單易用** - 拍照即可記錄

## 💰 價格策略分析

### 定價邏輯

#### 1. 免費版 10 次/月

```
目的：讓用戶體驗核心價值
計算：平均用戶每天記錄 1-2 次
      10 次 ≈ 5-10 天使用
      足夠培養習慣，但會產生需求
```

#### 2. Premium NT$ 149/月

```
對標分析：
├─ MyFitnessPal Premium: US$ 9.99/月 ≈ NT$ 320
├─ Foodvisor Premium: US$ 8.99/月 ≈ NT$ 290
├─ Lifesum Premium: US$ 6.99/月 ≈ NT$ 225
└─ Fooda Premium: NT$ 149/月 ✅ 最具競爭力

成本分析：
├─ AI API 成本: 300 次 × NT$ 0.008 = NT$ 2.4
├─ 雲端儲存: NT$ 1
├─ 平台抽成: NT$ 149 × 30% = NT$ 44.7
└─ 淨利潤: NT$ 149 - NT$ 48.1 = NT$ 100.9（67.7%）
```

#### 3. Premium Plus NT$ 299/月

```
價值定位：
├─ 無限 AI 識別（價值 > NT$ 500/月）
├─ 專業級分析工具
└─ VIP 服務體驗

風險控制：
├─ 設定合理使用限制（單日 50 次上限）
├─ 異常使用監控
└─ API 成本控制在 NT$ 50 以內
```
